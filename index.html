HTML

O que é HTML?

Passe o mouse para ver a resposta

Resposta:

Significa HyperText Markup Language (Linguagem de Marcação de Hipertexto). É a estrutura base de qualquer página na web, utilizada para definir e organizar os elementos do conteúdo.

CSS

O que é CSS?

Passe o mouse para ver a resposta

Resposta:

Significa Cascading Style Sheets (Folhas de Estilo em Cascata). É uma linguagem usada para estilizar a aparência e o layout de documentos HTML, cuidando de cores, fontes, espaçamentos e designs.

CSS / Layout

Para que serve o Flexbox?

Passe o mouse para ver a resposta

Resposta:

É um modelo de layout unidimensional projetado para alinhar, distribuir e organizar o espaço entre itens em um container, facilitando a criação de designs flexíveis e responsivos.

HTML Semântico

O que é uma tag semântica?

Passe o mouse para ver a resposta

Resposta:

É uma tag HTML que claramente descreve o seu significado e propósito tanto para o navegador quanto para o desenvolvedor (ex: <header>, <main>, <article>), melhorando a acessibilidade e o SEO.

Web Design

O que é Responsividade?

Passe o mouse para ver a resposta

Resposta:

É a técnica de design web que faz com que as páginas se adaptem de forma fluida e automática a qualquer tamanho de tela, garantindo uma excelente experiência de uso em computadores, tablets ou celulares.

CSS / Animação

Como funcionam as Transições?

Passe o mouse para ver a resposta

Resposta:

A propriedade transition do CSS permite alterar os valores das propriedades de forma suave e gradual durante uma mudança de estado (como um efeito de hover), definindo a duração e o ritmo dessa animação.
